Task 6 – Strong Password Creation & Strength Evaluation


Overview

This task demonstrates how to create strong, secure passwords and evaluate their effectiveness using online password strength checkers. By analyzing multiple password samples—from weak to highly secure—the task highlights the importance of complexity, length, randomness, and passphrases in cyber security.


Objectives

1.Understand the attributes of a strong password

2.Create sample passwords with varying complexity

3.Evaluate them using trusted password strength tools

4.Compare results and crack times

5.Document best practices for secure password creation

6.Study common password attacks and prevention techniques

Tools Used

1.passwordmeter.com

2.Kaspersky Password Checker

3.Bitwarden Strength Test

Key Steps

1.Created 5 passwords ranging from weak to excellent.

2.Tested each password using multiple checkers.

3.Recorded strength score, feedback, and estimated crack time.

4.Compared results in a structured table.

5.Researched common password attack methods.

6.Outlined security recommendations.

Key Learnings

1.Longer passwords are significantly harder to crack.

2.Randomness and diversity of characters improve security.

3.Passphrases provide strong protection with memorability.

4.MFA and password managers enhance overall cyber hygiene.

Files Included

1.Password Strength Analysis Report (PDF)

2.Realistic Password Strength Screenshot


Conclusion


This task strengthens understanding of password security fundamentals and highlights how small improvements in structure can drastically increase resistance against cyber attacks.
